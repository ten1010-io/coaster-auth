|===
| 상태 코드 | 용례

| 200 OK
| 요청을 성공적으로 처리함

| 400 Bad Request
| 잘못된 요청을 보낸 경우

| 401 Unauthorized
| 로그인이 필요한 경우

| 403 Forbidden
| 요청을 수행할 권한이 없는 경우

| 404 Not Found
| 요청한 리소스가 없음

| 409 Conflict
| 현재 서버의 리소스 상태와 충돌이 일어난 경우 +
예) 생성 요청시 요청한 리소스가 이미 존재, 업데이트 요청시 버전 불일치 등...

| 500 Internal Server Error
| 서버 에러
|===
